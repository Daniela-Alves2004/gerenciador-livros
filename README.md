# Projeto Full-Stack - Programação Web

Uma aplicação web full-stack desenvolvida para a disciplina de Programação Web Full-Stack.

## Como executar

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm start
```
